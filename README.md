# StudentManagerSystem
## 学生信息管理系统


### 运行环境
* JDK1.8编写
* Eclipse( Neon.3 Release (4.6.3)) windows下
* Tomcat v8.5
* Mysql 5.5


### 使用技术
* Java、 jsp、 JUnit4
* JavaScript & jQuery
* spring、 spring MVC、 Mybatis
* mybatis-generator逆向工具
* Mysql 数据库
* c3p0连接池
* log4j日志
* 封装Page类 用以数据库分页
* 使用bootstrap搭建前端展示界面


### 实现的功能
- 登录，用户名密码校验，异常处理。
- 实现了对学生的管理，完成了添加、删除、查询、修改学生信息等功能。
- 同时对权限进行简单控制，实现多级管理员身份。
